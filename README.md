# habit_tracker
The Habit Tracker is a simple and effective Python project that helps users build good habits and stay consistent. With this tool, you can add new habits, track which habits you completed today, and store all your progress safely in a file. It’s designed to be easy to use, beginner-friendly, and perfect for anyone who wants to improve their daily routine.

FEATURES

1. Add New Habits
Users can add habits they want to build.

2. Mark Habits as Completed
Track your daily achievements easily.

3. Auto Save to File
All habits are stored in a text file so your progress is not lost.

4. Duplicate Habit Prevention
Stops users from adding the same habit twice.

5. Simple Command-Line Interface
No complex setup — just run and use.

6. Easy to Customize
Great for learning file handling, loops, and basic project structuring

TECH STACK

Python 3.x

File Handling (Text Files)

Command Line Interface (CLI)

USE A MENU TO:

View your current habit list

Add a habit

Mark a habit as completed

Show today's progress

Show weekly summary
